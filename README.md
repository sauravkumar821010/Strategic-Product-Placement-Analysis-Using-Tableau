# 📌 Project Overview

Retail businesses rely on **product placement, pricing strategies, and promotional campaigns** to drive sales.
This project analyzes how these factors influence **customer purchasing behavior and retail performance** using interactive **Tableau dashboards and storytelling visualizations**.

The project converts raw retail data into **clear business insights** that help organizations make **data-driven decisions**.

---

# 🎯 Project Objectives

* 📊 Analyze **sales performance across product categories**
* 🏬 Understand the **impact of product display positions**
* 💰 Compare **product price with competitor pricing**
* 🎯 Evaluate **promotion effectiveness**
* 👥 Analyze **consumer segments**
* 📈 Provide **interactive dashboards for business insights**

The dataset includes:

* Product Category
* Product Price
* Competitor Price
* Promotion Status

* Sales Volume
* Consumer Segment
* Foot Traffic Level
